# Interpreting Results: C/IgG/IgM

- **C** = Negative.

- **C*IgM** = Positive.

- **C*IgG** = Positive.

- **C\*IgM\*IgG** = Positive.

- **Damaged/Lost** = Inconclusive.

- Any other combination = Invalid
